# Project Genesis

Project Genesis is a local-LLM world simulation and lore generation project.

It contains:

- Flask API/UI files for browsing generated civilizations and genealogy.
- Civilization, legend, origin myth, world map, and knowledge-base data.
- Dataset preparation and training scripts for a dedicated Genesis assistant.
- Training datasets used to build the local model behavior.

Large generated model artifacts are intentionally excluded from this repository:

- `genesis_merged_model/`
- `genesis_lora_model/`
- `genesis_model_output/`
- `*.safetensors`, `*.pt`, `*.pth`, `*.bin`

The original local working copy may still contain those files.
