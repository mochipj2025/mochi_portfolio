# リーダーシップの新しい概念 (Shinsei Leaderhood)

## 定義
リーダーシップは、集団の共通価値観と目標に基づく相互作用によって形成される概念である。individualsの内在的性質や外部的要因が絡み合い、collective values and goalsを反映したものである。

## 意義
この新しい概念では、リーダーシップは集団の精神状態や価値観に基づくものであり、特定の個体に固着することなく、相互作用によって形成される。したがって、この概念では、リーダーシップを天性か否かに関わらず、その実態は集団の共通価値観と目標に依存している。

この新しい概念は、権力や富など社会的影響力を (-10) 評価する一方、信仰などの精神的な影響力を (0) 評価し、権力 (+5) の評価を下げることを示す。
