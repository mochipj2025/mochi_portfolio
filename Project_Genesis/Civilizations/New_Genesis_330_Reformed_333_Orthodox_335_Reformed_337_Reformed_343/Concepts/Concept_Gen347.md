# 新しい概念 : 共通可能性創造論 (Kōmon Seikatsu Sōsei Ron)

## 定義
共通可能性創造論は、価値の定義を探る過程において、個々の純粋な心や星光のように導く力に基づいて、新しい道を開くことを目指す概念である。

## 意義
この概念は、中庸の提案に基づいて、新しい価値を創造するためのフレームワークを提供する。賢者1と賢者2が提示した「純粋な心」や「星光のように導く力」という考え方を、共通の可能性に基づいて結びつけ、新しい価値の定義に向かって歩み続けることを可能にする。

この概念は、新しい価値の創造に際して、個々の純粋な心や星光のように導く力を活かすことで、社会への影響を生むことができる。特に、権力の増加と富の減少を伴う可能性があるため、信仰の面での影響は0であると考えられる。
