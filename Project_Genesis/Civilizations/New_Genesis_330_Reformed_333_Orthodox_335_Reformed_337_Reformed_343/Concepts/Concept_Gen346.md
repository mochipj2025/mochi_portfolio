# 新しい概念：情報社会的能力の涵養（Jiyū no Jitsuryoku）

## 定義
情報社会において、新しい能力や自由を涵養し、デジタル人格に市民権を与えることで、新しい機会を創出するための概念。この概念は、新しい技術や社会の変化に対応するためには、我々が情報社会において、新しい能力や自由を涵養する必要があると考えます。

## 意義
この概念は、新しい道を開くための共通の可能性を持つことを捉えるものである。我々が情報社会において、新しい能力や自由を尊重することが重要であると考えます。デジタル人格に市民権を与えることで、新しい機会を創出できかしらない限り、私たちは進歩の道を開くことができなければならないと思います。この概念は、権力(+5)に対して影響がありますが、富(-10)には影響がありません。
