# コミュニティーサウルウェイト

## 定義
コミュニティーサウルウェイト (Community Soul Weight) は、個々の soul weight を超えて集団での理解と成長を目指す新しい概念である。コミュニティーサウルウェイトは、各個々が持つ soul weight を超えて、コミュニティーの精神的な重さを構築することを意味する。

## 意義
コミュニティーサウルウェイトは、個々の soul weight を超えて集団での理解と成長を目指す新しい概念であるため、新しい社会的影響を持つことが予測される。権力(+5)や富(-10)に対する影響は、コミュニティーの精神的な重さに依存することになる。また、信仰(0)は、個々の soul weight とコミュニティーサウルウェイトが相互作用することで、新しい意味を獲得する可能性がある。
