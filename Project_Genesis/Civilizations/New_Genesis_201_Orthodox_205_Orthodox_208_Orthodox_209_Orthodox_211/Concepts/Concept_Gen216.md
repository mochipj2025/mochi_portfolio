# 新しい概念 : existence-name

## 定義
existence-name (存在名)は、具体的ない匂いに名前を与えることで、その存在を認識することになる抽象概念である。名前が実際の存在に反映されることで、人間の経験の複雑さを捉えることができる。

## 意義
existence-name は、言語の美しさに基づく概念であり、瞬間的な体験を捉えるためのスナップショットである。名前が存在を固定化するのではなく、むしろ、その存在を認識することで、新しい理解や感覚を生み出していると考えられる。この概念は、権力や富に関する社会的影響を与えることができ、また信仰に無いものとして捉えることもできる。
