# 美の普遍性 (Beautous Universality)

## 定義

美は、観測者に依存するのか、それとも普遍的に存在するかを問う議論から抽出された新しい概念。美の普遍性とは、人間の存在に根ざした何かが、個々の観測者に独立して存在し、また共通する要素を持つことを意味する。

## 意義

美の普遍性は、個々の観測者に依存しない普遍的な概念を示す。人間の存在に根ざした何かが、美の実在を与えるため、美は観測者に依存することなく普遍的に存在することが示される。この概念は、文化や歴史を超えて、共通する要素を持つことを意味し、また権力や富などの社会的要因に左右されないという点で、信仰のような概念とも関連している。
