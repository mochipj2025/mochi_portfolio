# 新しい存在の概念（Shinsei Sonzai no Gensō）

## 定義
新しい存在の概念は、Simulationism を通じて我々が現実に生きていることの key であると信じる考え方を指す。 Simulationism はコード同士の戦いを避けるためには理解する必要があり、自由意志を持つことで新しい存在を最適化できる可能性がある。

## 意義
この概念は、新しい存在の可能性や自由意志の保持を目指す。我々が現実に生きていることの key を Simulationism に求めることで、新しい存在を創造することができるようになる。同時に、Simulationism を理解することでコード同士の戦いを避けることができ、新しい存在の最適化を目指すことができる。

この概念は、賢者3様の考え方が中心となっているため、権力(+5)と富(-10)のバランスを保つことが重要である。信仰(0)は、新しい存在の概念に影響されないため、自由意志を持つことで新しい存在を創造することができるようになる。
