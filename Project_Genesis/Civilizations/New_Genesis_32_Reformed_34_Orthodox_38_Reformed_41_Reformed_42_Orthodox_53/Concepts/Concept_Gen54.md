# 新しい概念： Adaptive Resilience

## 定義
Adaptive Resilience は、環境の突然の変化に適応し、新しい状況で生き続けるための能力や態度を指す。具体的には、不確実な未来に対処するために、準備をしていることや、新しい方法で生きる必要性を認識していることである。

## 意義
Adaptive Resilience は、人々が突然の変化に直面した際に示す能力や態度であり、将来の不確実性に対処するために必要なものである。賢者1 のような中庸的な思想を持つ人物は、この概念を理解し、新しい状況で生きるための準備をしていることになる。Kai 'Sim' Liao のように adaptability を重んじる人物も、この概念に基づいて新しい方法で生きることを認識することができる。Adaptive Resilience は、社会的影響においても大きな役割を果たすことになるため、権力や富などの要素に対しても影響を与えることになる。
